# weather-now

A simple React app that allows users to quickly check the **current weather** in any city using the **Open-Meteo API**.
--- Features
- Search weather by **city name**
- Shows:
  -  Temperature (°C)
  -  Wind speed (km/h)
  -  Wind direction (°)
- Error handling for unknown cities
- Built with **React + Vite**

--- Tech Stack
- **React 18**
- **Vite** (fast build tool)
- **JavaScript (ES6+)**
- **Open-Meteo API** (free weather API)
